# 抽籤分組助手 (Lucky Draw & Grouping Assistant)

一個純前端的 Web 工具，旨在幫助使用者快速進行名單抽籤與自動分組。適合辦公室、教室或活動現場使用。

## 🌟 主要功能

### 1. 名單管理
- **快速輸入**：支持大型文本框，一行一個姓名。
- **偵測重複**：自動標記重複姓名，並提供一鍵移除功能。
- **模擬數據**：內建模擬名單，方便快速測試。
- **持久化儲存**：使用 `localStorage` 自動保存名單、抽籤結果與分組狀態，重新整理也不會消失。

### 2. 隨機抽籤 (Target Draw)
- **自定義人數**：可設定一次抽取的得獎者數量。
- **重複設定**：可選擇「允許重複抽取」或「不重複抽取」。
- **動態效果**：內建 2 秒的「老虎機式」滾動動畫，增加緊張氣氛。
- **剩餘人數統計**：即時顯示目前池中還剩下多少人。

### 3. 自動分組 (Auto Grouping)
- **靈活分組**：根據「每組人數」自動計算組數。
- **視覺化呈現**：分組結果以卡片方式排列，清晰明瞭。
- **CSV 下載**：分組完成後，可一鍵導出為 CSV 檔案（包含 UTF-8 BOM，Excel 開啟不亂碼）。

## 🛠️ 技術細節
- **架構**：單一 HTML 檔案，不需額外引入外部 JS/CSS 套件。
- **語言**：HTML5, CSS3, Vanilla JavaScript (ES6+)。
- **字體**：使用 Google Fonts (Inter, Noto Sans TC)。
- **響應式**：支援桌面與行動裝置顯示。

## 🚀 如何使用
1. 直接用瀏覽器開啟 `index.html`。
2. 在左側輸入名單。
3. 選擇上方「名單抽籤」或下方「自動分組」功能區進行操作。
